# かんたん更新ガイド（マツバ工房さま向け）

ブラウザだけで、写真の差し替えや文章の修正ができます。
むずかしい知識は不要です。不安なときは無理せずご相談ください。

―――――――――――――――――――――――――

## このサイトについて

- **公開URL** … <https://arashooo.github.io/matsuba-kobo/>
- **中身（データ）** … GitHub の `matsuba-kobo` に保管
  <https://github.com/Arashooo/matsuba-kobo>
- **仕組み** … GitHub上のファイルを変更すると、**約1分後に自動でサイトへ反映**されます。

## 更新するための準備（最初だけ）

GitHubにログインし、このデータを編集できる権限が必要です。
編集したい方は、管理者にGitHubのアカウント名を伝え、
「Collaborator（共同編集者）」に追加してもらってください。

―――――――――――――――――――――――――

## 1. 商品写真を差し替える（いちばん多い更新）

1. <https://github.com/Arashooo/matsuba-kobo/tree/main/assets/products> を開く
2. 「**Add file**」→「**Upload files**」をクリック
3. 新しい写真を、差し替えたい写真と**同じファイル名**にしてドラッグ＆ドロップ

   | ファイル名 | 場所 |
   |---|---|
   | `haizara.jpg` | 携帯灰皿 |
   | `saikoro.jpg` | サイコロ |
   | `bell.jpg` | ガーディアンベル |
   | `tamago.jpg` | しんちゅうたまご（「真鍮について」の写真も同じ） |
   | `bangle.jpg` | バングル |
   | `koma.jpg` | 駒 |

4. 下の「**Commit changes**」を押す → **約1分でサイトに反映**
   - 写真はタテ長〜正方形に近いものがきれいに収まります。

―――――――――――――――――――――――――

## 2. 文章を直す

1. <https://github.com/Arashooo/matsuba-kobo/blob/main/index.html> を開く
2. 右上の**鉛筆マーク（Edit）**をクリック
3. 直したい文章を書き換える
   - ⚠️ `< >` で囲まれた部分（タグ）は消さない。**文字の部分だけ**書き換える。
4. 「**Commit changes**」を押す → 約1分で反映

―――――――――――――――――――――――――

## 3. パソコンのファイルで直したい場合（上級者向け）

- 「**Code**」→「**Download ZIP**」でデータ一式を取得して編集 → 再アップロード
- git が使える方は `clone` して編集 → `push`
