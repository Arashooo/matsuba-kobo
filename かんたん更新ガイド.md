# かんたん更新ガイド（マツバ工房さま向け）

このサイトは **HTMLファイル** でできています。むずかしい知識がなくても、
文字の打ち替えと写真の差し替えくらいなら、ご自身でできます。
（不安なときは、無理せずご相談ください）

―――――――――――――――――――――――――

## 1. 文章を直したいとき

1. `index.html` を「メモ帳」や「テキストエディット」で開きます
   （右クリック →「プログラムから開く」→ テキストエディット）
2. 直したい文章を探して、書き換えて、保存するだけ
3. ブラウザで `index.html` を開くと反映されています

> ⚠️ `<　>` で囲まれた部分（タグ）は消さないでください。
> 文字の部分だけ書き換えれば大丈夫です。

―――――――――――――――――――――――――

## 2. 商品写真を差し替えたいとき

写真は `assets/products/` フォルダに入っています。

| ファイル名 | 今の写真 |
|---|---|
| `haizara.jpg` | 携帯灰皿 |
| `saikoro.jpg` | サイコロ |
| `bell.jpg` | ガーディアンベル |
| `tamago.jpg` | しんちゅうたまご |
| `bangle.jpg` | バングル |
| `koma.jpg` | おこま |

**差し替え方：** 新しい写真を **同じ名前** にして、このフォルダの古いファイルに
上書きするだけ。サイト側の書き換えは不要です。
（写真はタテ長・正方形に近いものがきれいに収まります）

―――――――――――――――――――――――――

## 3. 新商品を増やしたいとき

`index.html` の中の「作品（works）」エリアにある、商品1つ分のかたまり
（`<article class="work-card">` から `</article>` まで）をコピーして貼り付け、
写真名・商品名・説明文を変えれば増やせます。
※ ここだけ少しコツがいるので、迷ったらご相談ください。

―――――――――――――――――――――――――

## 4. 公開した内容を更新するには

- **Netlifyに登録して使っている場合** … 管理画面に新しいフォルダをドラッグするだけで上書き更新
- 登録なしの一時URLの場合 … もう一度 <https://app.netlify.com/drop> にドラッグすると新URLが出ます
