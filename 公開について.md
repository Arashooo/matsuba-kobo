# 公開について（現在の構成）

## いまの状態

- **公開URL** … <https://arashooo.github.io/matsuba-kobo/>
- **ホスティング** … GitHub Pages（**無料**）
- **データ（リポジトリ）** … <https://github.com/Arashooo/matsuba-kobo>
- **更新方法** … リポジトリのファイルを変更すると約1分で自動反映
  （→ `かんたん更新ガイド.md`）

すでに公開済みなので、URLをそのまま共有・SNSに掲載できます。

## SNS・各所への掲載

- Instagram / X のプロフィールリンク
- 楽天ショップの自己紹介
- 名刺・フライヤー など

## 独自ドメインを使いたい場合（任意・あとからでOK）

1. ドメインを取得（お名前.com など、年1,000円程度）
2. GitHubリポジトリの **Settings → Pages → Custom domain** にドメインを入力
3. ドメイン側のDNSに `CNAME` で `arashooo.github.io` を設定
4. 設定後、HTTPS（鍵マーク）が自動で有効になります

## 別のサービスへ移したい場合

静的サイトなので、データ一式をそのまま以下に置いても動きます。

- Netlify / Cloudflare Pages / Vercel（フォルダをドラッグ＆ドロップ）
- レンタルサーバー（FTPでアップロード）

## 補足

- `matsuba_zine/`（撮影した元データ）と `assets/zine/` は公開には不要なため、
  リポジトリには含めていません（手元にのみ保管）。
