# マツバ工房 ウェブサイト

名古屋の真鍮ハンドメイド工房「マツバ工房（有限会社マツバ）」の紹介サイトです。
ビルドツール不要の静的サイト（HTML / CSS / JavaScript）です。

- **公開URL** … <https://arashooo.github.io/matsuba-kobo/>
- **ホスティング** … GitHub Pages（無料）／ファイルを更新すると約1分で自動反映

## まず読むもの（用途別）

| やりたいこと | 読むファイル |
|---|---|
| 写真・文章を自分で更新したい | **`かんたん更新ガイド.md`** |
| 公開の仕組み・独自ドメイン・移行 | **`公開について.md`** |
| デザインの方針（色・書体・余白） | **`DESIGN.md`** |

## ファイル構成

```
matsuba_web/
├─ index.html              … ページ本体（全セクション）
├─ styles.css              … デザイン・レイアウト
├─ script.js              … ナビ・スクロール演出・お問い合わせフォーム
├─ assets/products/        … 作品写真
├─ DESIGN.md               … デザイン指針（デザイントークン）
├─ かんたん更新ガイド.md   … 工房向け・自己メンテ手順
├─ 公開について.md         … 公開構成・ドメイン・移行
└─ README.md               … このファイル
```

> 撮影元データ `matsuba_zine/` と未使用の `assets/zine/` は、公開リポジトリには含めていません（`.gitignore` で除外）。

## ローカルで見る

`index.html` をダブルクリックでも表示できますが、ローカルサーバー経由が確実です。

```bash
cd matsuba_web
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

## 作品写真の差し替え

写真は `assets/products/` にあります。**同じファイル名で上書き**すれば、HTMLの編集は不要です。

| ファイル名 | 場所 |
|---|---|
| `haizara.jpg` | 携帯灰皿 |
| `saikoro.jpg` | サイコロ |
| `bell.jpg` | ガーディアンベル |
| `tamago.jpg` | しんちゅうたまご（「真鍮について」の写真も同じ） |
| `bangle.jpg` | バングル |
| `koma.jpg` | 駒 |

縦長〜正方形に近い写真がきれいに収まります（カードは表示時に中央でトリミングされます）。

## お問い合わせフォーム

`script.js` は **mailto 方式**です。送信を押すと、入力内容が入った状態で
メール作成画面（宛先 `mail@matsuba.info`）が開きます（サーバー不要）。

ページ内で完結し指定アドレスへ自動送信する正式なフォームにしたい場合は、
[Formspree](https://formspree.io/) などのフォームサービスに繋ぎ替えられます。
