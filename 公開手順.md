# 公開手順（無料・5分）

このサイトは特別なサーバーがなくても、ファイルを置くだけで公開できます。
いちばん簡単な **Netlify Drop（ネットリファイ・ドロップ）** の手順です。

## 事前に：アップロードするフォルダを軽くする
`matsuba_web` フォルダの中の **`matsuba_zine` フォルダ（撮影した元写真）は公開に不要**です。
アップロード前に、`matsuba_web` を複製して、複製のほうから `matsuba_zine` を削除すると軽くなります。
（必要な写真は `assets/` 内にコピー済みなので消しても表示は変わりません）

## 手順
1. ブラウザで <https://app.netlify.com/drop> を開く
2. `matsuba_web` フォルダを、画面の点線エリアに **ドラッグ＆ドロップ**
3. 数十秒待つと、`https://〇〇〇.netlify.app` のような **公開URL** が発行されます
4. そのURLをスマホで開いて確認 → そのまま相手に共有できます

> アカウント登録なしでも一時公開できます。
> 登録（無料）すると、URLの名前変更・更新・独自ドメイン接続ができます。

## 独自ドメインを使いたい場合（任意・後からでOK）
- お名前.com / ムームードメイン等で年1,000円程度のドメインを取得
- Netlifyの「Domain settings」から接続
- まずは `.netlify.app` のURLで十分。InstagramのプロフィールやX、楽天の自己紹介に貼れます

## 他の選択肢
- **Cloudflare Pages** / **Vercel** … 同様に無料・ドラッグ＆ドロップ対応
- **レンタルサーバー（FTP）** … `matsuba_web` の中身を `public_html` にアップロード

## 公開後にやると良いこと
- InstagramプロフィールのリンクをこのサイトURLに
- 楽天・X のプロフィールにもURLを記載
- Google Search Console に登録すると検索に出やすくなります（任意）
